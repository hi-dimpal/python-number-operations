# Number Operations in Python

This is a menu-driven Python program that performs basic number operations.

## Features
- Convert negative number to positive
- Reverse a number
- Check palindrome
- Check even or odd

## Concepts Used
- Functions
- While loop
- Conditional statements
- Menu-driven programming

## How to Run
1. Clone the repository
2. Run the Python file using:
   python number_operations.py
